Copy and paste these commands one by one into your Terminal:

1. First, set the Android SDK path:
export ANDROID_HOME=/Users/varunkumar/Library/Android/sdk
export PATH=$PATH:$ANDROID_HOME/cmdline-tools/latest/bin:$ANDROID_HOME/platform-tools

2. Check current emulators:
avdmanager list avd

3. Delete old tablet emulators:
avdmanager delete avd -n Pixel_Tablet_API_34
avdmanager delete avd -n Simple_Tablet

4. Create new Pixel Tablet emulator:
avdmanager create avd -n Pixel_Tablet_API_34 -k "system-images;android-34;google_apis;x86_64" -d "pixel_tablet"

5. Create new Simple Tablet emulator:
avdmanager create avd -n Simple_Tablet_API_34 -k "system-images;android-34;google_apis;x86_64" -d "10.1in WXGA (Tablet)"

6. Verify they were created:
avdmanager list avd

7. Launch the emulators:
flutter emulators --launch Pixel_Tablet_API_34
flutter emulators --launch Simple_Tablet_API_34

8. Install your APK:
flutter install --device-id=<emulator_id>
